¿Quiénes somos?

Somos dos mejores amigos apasionados por la tecnología y el desarrollo web. Nuestra amistad y amor por la programación nos inspiraron a crear una aplicación que reflejara lo aprendido y ayudara a otros a iniciar su camino en Ingeniería en Software.

La idea surgió tras un cuatrimestre intenso en el que aprendimos Angular. Decidimos unir fuerzas y conocimientos para desarrollar una plataforma de cursos que abordara las bases fundamentales de nuestra carrera.

Conoce al equipo de desarrollo de esta plataforma de ayuda a la comunidad estudiantil de Ingenieria de Software.

Apoya nuestro proyecto

Gracias a tu apoyo, podemos seguir ofreciendo contenido gratuito de calidad.

Este proyecto lo hemos desarrollado a base de todo el conocimiento, que hemos adquirido durante 5 cuatrimestres, usamos Frameworks modernos para el desarrollo de la aplicación web, algunas tecnologías que usamos fueron:

Angular y Node Js

+

Este Framework nos fue de mucha ayuda para realizar el Front-End de esta aplicación web. En conjunto con npm y Node.js.

AWS

Esta tecnología nos ha sido de mucha ayuda para nuestro servidor web, donde hemos desplegado nuestra API REST.

Python (FastAPI)

Esta tecnología nos ha sido de mucha ayuda para el desarrollo del Back-End utilizando FastAPI.

MySQL

Esta tecnología la hemos usado para gestionar nuestra base de datos de comentarios y sugerencias.

Linux

Este sistema operativo nos ha ayudado a desarrollar y gestionar de manera organizada y segura.

GitHub

GitHub ha sido esencial para la gestión de nuestro código fuente y trabajo colaborativo.
