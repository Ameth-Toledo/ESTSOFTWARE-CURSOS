¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado utilizado para estructurar el contenido en la web. Permite crear elementos como títulos, párrafos, imágenes y enlaces.

  • <h1></h1>: Títulos principales.
  • <p></p>: Párrafos.
  • <a href="url"></a>: Enlaces.
  • <img src="url" alt="texto">: Imágenes.

¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje utilizado para estilizar el contenido HTML. Permite controlar colores, fuentes, espaciados y diseños responsivos.

  • Colores: color: red;
  • Fuentes: font-family: Arial;
  • Margen: margin: 10px;
  • Diseños: display: flex;

¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite agregar interactividad a las páginas web, como botones funcionales, animaciones y validaciones de formularios.

  • Alertas: alert('¡Hola mundo!');
  • Eventos: document.getElementById('boton').addEventListener('click', funcion);
  • Manipulación DOM: document.querySelector('h1').innerText = 'Nuevo título';
