Comando Clear

El comando clear es utilizado en Linux para limpiar la terminal. Esto es útil cuando queremos despejar la pantalla de texto previamente ejecutado, facilitando la visualización de nuevos comandos o salidas.

Al ejecutar este comando, el contenido de la terminal se oculta, pero no se elimina. La información anterior aún puede ser visible si se desplaza hacia arriba (dependiendo del terminal que estés usando).

Comando LS

El comando ls se utiliza para listar los archivos y directorios en el directorio actual. Este comando es muy útil para explorar el contenido de tu sistema de archivos.

Puedes combinar ls con opciones para obtener más información, como:

  • ls -l: Muestra los archivos en formato de lista detallada.
  • ls -a: Incluye archivos ocultos en el listado.

Comando CD

El comando cd (change directory) permite navegar entre directorios en el sistema de archivos. Es una herramienta esencial para moverte dentro de la estructura de directorios de Linux.

Algunos ejemplos comunes son:

  • cd /: Cambia al directorio raíz.
  • cd ..: Sube un nivel al directorio padre.
  • cd ~/Documentos: Cambia al directorio "Documentos" dentro de tu carpeta personal.
