Comando TOUCH

El comando touch se utiliza para crear archivos vacíos o actualizar la fecha y hora de modificación de archivos existentes. Es una herramienta básica pero muy útil para crear rápidamente archivos nuevos.

Ejemplos prácticos:

  • touch archivo.txt: Crea un archivo vacío llamado "archivo.txt".
  • touch archivo1 archivo2: Crea varios archivos vacíos simultáneamente.

Comando CAT

El comando cat (concatenate) se utiliza para mostrar el contenido de un archivo en la terminal, combinar archivos o crear archivos nuevos desde la terminal.

Ejemplos prácticos:

  • cat archivo.txt: Muestra el contenido del archivo "archivo.txt".
  • cat archivo1 archivo2 > combinado.txt: Combina "archivo1" y "archivo2" en un nuevo archivo llamado "combinado.txt".

Comando NANO

El comando nano abre un editor de texto simple y fácil de usar directamente en la terminal. Es ideal para editar archivos rápidamente.

Ejemplos prácticos:

  • nano archivo.txt: Abre el archivo "archivo.txt" en el editor nano.
  • Dentro de nano, usa Ctrl + O para guardar y Ctrl + X para salir.

Comando ECHO

El comando echo se utiliza para imprimir texto o variables en la terminal. También puede redirigir la salida a archivos.

Ejemplos prácticos:

  • echo "Hola Mundo": Imprime "Hola Mundo" en la terminal.
  • echo "Hola Mundo" > archivo.txt: Crea un archivo "archivo.txt" con el texto "Hola Mundo".
