Comando MKDIR

El comando mkdir (make directory) se utiliza para crear nuevos directorios en Linux. Es útil para organizar tus archivos y mantener una estructura de directorios bien definida.

Algunos ejemplos prácticos:

  • mkdir nuevo_directorio: Crea un directorio llamado "nuevo_directorio".
  • mkdir -p padre/hijo: Crea una estructura de directorios anidada, como "padre" y dentro de él "hijo".

Comando RMDIR

El comando rmdir (remove directory) se utiliza para eliminar directorios vacíos en Linux. Si un directorio contiene archivos, no puede ser eliminado con este comando.

Ejemplo práctico:

  • rmdir viejo_directorio: Elimina el directorio "viejo_directorio" si está vacío.

Comando RM

El comando rm (remove) se utiliza para eliminar archivos o directorios en Linux. Por defecto, no elimina directorios si contienen archivos.

Ejemplo práctico:

  • rm archivo.txt: Elimina el archivo "archivo.txt".
  • rm -i archivo.txt: Solicita confirmación antes de eliminar "archivo.txt".

Comando RM -R

El comando rm -r (remove recursively) elimina directorios y su contenido de forma recursiva. Es una herramienta poderosa, pero debes usarla con cuidado, ya que puede eliminar grandes cantidades de datos rápidamente.

Ejemplo práctico:

  • rm -r carpeta: Elimina la carpeta "carpeta" y todo su contenido.
  • rm -rf carpeta: Elimina la carpeta "carpeta" sin solicitar confirmación, incluso si contiene archivos protegidos.
